"🔥"		 Resposta amigável. Não quebra. Não tenta classificar como transação. edge
EC4 Valor com vírgula brasileira
"1.500,50"		 Parseado como R$ 1.500,50 (não R$ 150.050). Confirmação mostra valor correto. edge
EC5 Valor com abreviação "k"
"25k"		 Parseado como R$ 25.000 (não R$ 25). Regra Nk = N × 1000. edge

(pending de "Botox", manda "Aluguel 3000")		 Descarta pending anterior e processa novo comando. Não mistura dados. edgestate
